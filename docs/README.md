# Docs

This folder keeps the repo-level notes out of the root while preserving a
clear front door for active work.

## Layout

- [`campaign`](campaign): campaign plans, opcode conclusion notes, and the
  milestone archive that supports [`../RE_CAMPAIGN_MEMORY.md`](../RE_CAMPAIGN_MEMORY.md)
- [`workflows`](workflows): session instructions, strategy notes, and
  tool-specific workflows
- [`research`](research): broader analysis notes that are useful context but
  are not the active campaign ledger
